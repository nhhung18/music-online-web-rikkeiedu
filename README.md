# Music Online

Music Online là hệ thống nghe nhạc trực tuyến được phát triển bởi đội ngũ sinh viên thực tập của trường Đại học Thăng Long trong kỳ thực tập với đơn vị RikkeiEdu thuộc RikkeiSoft. Đây là source code hoàn chỉnh và đã hoàn thành bảo vệ với đơn vị thực tập xong.

## Công nghệ sử dụng
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL
- Gradle

## Công cụ
- Github
- Cloudinary

## Chức năng chính
- Đăng nhập, Đăng ký, phân quyền người dùng
- Nghe nhạc trực tuyến
- Mua gói Membership
- Quản lý Album, Banner, Genre, Playlist, Favorite
- Download nhạc, commment, đánh giá
